# spark
Create Lightning apps with no build configuration.
- Getting Started - How to create a new app.
- User Guide - How to develop apps bootstrapped with Spark.

## Requirements
- SFDX
- Node.js

## Quick Overview
```
npm install -g spark

spark my-app
cd my-app/
npm preview:new
```
